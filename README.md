# spark_test
Данная джоба производит агрегация метрик по времени
Пример входного датасета лежит в src/main/resources/dataset.csv
 
## Собрка
Для собрки jar файла перейдите в директорию проекта и выполните следующую команду
```
mvn clean package
```

## Запуск
Для запуска jar файла необходимо после сборки из директории проекта запустить следующую команду
```
java -jar target/spark_test-jar-with-dependencies.jar datasetPath minutes
```
где  
datasetPath - путь до датасета с входными данными (метриками)  
minutes - значение минут, по которому будет производится агрегирование

## Результат
Результирующий датасет помещает в csv файл в директорию result/
